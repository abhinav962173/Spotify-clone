## 📌 About

This project is a front-end clone of Spotify's UI.  
The goal was to replicate the look and feel of Spotify using only HTML and CSS — no frameworks, no JavaScript libraries.

---

## ✨ Features

- 🎵 Music player UI with album art
- ▶️ Play / Pause button layout
- 📱 Responsive design — works on all screen sizes
- 🎨 Dark theme matching Spotify's aesthetic
- 🖼️ Assets folder for images and icons

---
